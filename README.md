# form-docs

[![Deploy](https://github.com/form-dev/form-docs/actions/workflows/deploy.yml/badge.svg)](https://github.com/form-dev/form-docs/actions/workflows/deploy.yml)

This repository automatically builds and publishes
the [FORM documentation](https://form-dev.github.io/form-docs/)
using sources from the [FORM repository](https://github.com/form-dev/form).

- [Latest development reference manual](https://form-dev.github.io/form-docs/master/manual)
- [Stable reference manual](https://form-dev.github.io/form-docs/stable/manual)
